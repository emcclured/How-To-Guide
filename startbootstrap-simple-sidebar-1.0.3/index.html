What is this guide about?

In this how-to-guide, we will apply PDO to the creation of a video database. We will cover the basic implementations for PDO in this database example, compare it to MySQLi format where there are significant differences, and show the end result database.

Brief information on PDO is provided in this guide, but the majority of the information will be left for other more indepth tutorials available online. A few links is provided on the 'Overview' page at the end, which will give you more background information on PDO. A link to the video database is also given. Here we will focus on applying PDO to a specific example--a video database.


What is PDO?

PDO stands for PHP Data Object. PDO is a way for the PHP programming langugage to speak with the MySQL database. More specifically, PDO is a type of application programming interface (API) that is used to connect PHP programs to a MySQL database. Altogether, there are three API's available to connect PHP to MySQL databases--the other two are MySQL and MySQLi.


Why use PDO?

There are three main benefits of PDO:it provides a robust API, it allows use of multiple different databases, and PDO offers prepared statement ability. Translation as to what this will do for you: flexible, elegant, and secure code.

If you want to be able to use more than one database, PDO is the way to go. The added level of security is also a plus.
