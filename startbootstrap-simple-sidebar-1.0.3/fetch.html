At the while loop portion of the code, notice that PDO::FETCH_ASSOC is the type of fetch that was chosen here. This means that an array is returned, indexed bu column name for our video database fetch. There are other types of fetch that you can implement in PDO, Additional Resources section of the 'Overiew' page contains links that will provide more information on these.

PDO Fetch Options:
  • PDO::FETCH_BOTH
  • PDO::FETCH_BOUND
  • PDO::FETCH_CLASS
  • PDO::FETCH_INTO
  • PDO::FETCH_LAZY
  • PDO::FETCH_NAMED
  • PDO::FETCH_NUM
  • PDO::FETCH_OBJ
