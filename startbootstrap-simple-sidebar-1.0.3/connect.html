PDO Error Modes

Note that in the PDO connection, the PDO::ERRMODE_EXCEPTION. Exception error mode will send out exceptions as they occur, and has been modifed from the default, PDO::ERRMODE_SILENT. Manual error fetching with quieries would need to be carried out with the silent mode option. Lastly, there is an PDO::ERRMODE_WARNING option with PDO, which sends out an error code and warning message.

PDO Error Mode Options:
  • PDO::ERRMODE_WARNING
  • PDO::ERRMODE_SILENT
  • PDO::ERRMODE_EXCEPTION
